- Update polices
- Receive/send RBTC
- Receive/send ERC20
- Receive/send ERC721
- Call contract
- Off-chain signatures
- Reject and cancel tx

Read the docs: [developers.rsk.co](https://developers.rsk.co/rif/multisig/)

## Run for development

Install dependencies:

```
npm i
```

### Compile with TS

```
npm run build
```

Watch:

```
npm run build:watch
```

### Run the local network

```
npm run network:local
```

### Run the script

```
npm run start
```

Watch:

```
npm run build:start
```

### Run the script for development

```
npm run dev
```
